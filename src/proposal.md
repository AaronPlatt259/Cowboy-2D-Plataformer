# Cowboy 2D platform level

## Repository
<https://github.com/AaronPlatt259/Cowboy-2D-Plataformer.git>

## Description
Small level of a 2d platform game were you can customize the aesthetics (and hopefully the obstacles)  of the level with the options given


## Features

- Aesthetic and  enemy customization
	- Pygame lets you make something happen once you hit a key or click with the mouse, I will make the level customizable by using this things(if you press a key or select something with your mouse, you can add that to the level)

-  Menu for customization
	-I will use the pygame MOUSECLICK and KEYDOWN to make a simple menu for the customization to be easily understandable (so you only have to click a box to change something like the sky for example)

- Level with character
	-I will make a character that has to go through simple obstacles and enemies to get to the end of the level
